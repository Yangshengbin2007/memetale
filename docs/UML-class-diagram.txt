================================================================================
Key Relationships:
================================================================================

1. Main → SceneManager → Scene (interface)
   - Main creates SceneManager
   - SceneManager manages Scene instances

2. Scene implementations:
   - StartScene, ChapterOneScene, MiniGameCollectionScene all implement Scene
   - All extend JPanel for Swing UI

3. GameState (Singleton) → StoryState
   - Global access point for game state
   - StoryState contains game progress

4. StoryState → DialogueRecord
   - History stored as List of DialogueRecord

5. SaveLoad → StoryState
   - Serialization/deserialization for save/load

6. ChapterOneData
   - Static utility class with dialogue data
   - Used by ChapterOneScene
